Flask Portfolio Website

Персональный портфолио-сайт на Flask, предназначенный для представления проектов, навыков и контактной информации.
Проект построен с использованием Flask, Bootstrap и шаблонов HTML, с разделением backend-логики и статических ресурсов.

📌 Описание

Этот проект — минималистичное и расширяемое веб-портфолио, которое можно использовать как:

личный сайт разработчика,

демонстрацию навыков Flask / Python,

основу для дальнейшего развития (блог, админка, CMS).

Сайт адаптивен и корректно отображается на мобильных устройствах.

🚀 Возможности

🌐 Веб-приложение на Flask

🎨 Адаптивный интерфейс (Bootstrap)

🧩 HTML-шаблоны с поддержкой Jinja2

📁 Статические ресурсы (CSS, JS, изображения)

⚙️ Установка и запуск
1️⃣ Клонирование репозитория
git clone https://github.com/<your_username>/<repo_name>.git
cd <repo_name>

2️⃣ Установка зависимостей

Рекомендуется использовать виртуальное окружение:

python -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate
pip install -r requirements.txt

3️⃣ Запуск приложения
python app.py


По умолчанию сайт будет доступен по адресу:

http://127.0.0.1:5000

🧠 Как это работает

Flask отвечает за маршрутизацию и отдачу страниц

HTML (Jinja2) используется для шаблонов

static/ содержит все стили, скрипты и изображения

Bootstrap обеспечивает адаптивную вёрстку

Проект легко расширяется:

добавлением новых страниц,

подключением базы данных,

внедрением формы обратной связи,

деплоем на VPS или PaaS.

🌍 Деплой (идеи)

Проект можно развернуть на:

Render

Railway

Fly.io

VPS (Nginx + Gunicorn)

🛡 Безопасность

Проект не содержит секретных данных

Подходит для публичного размещения

Рекомендуется вынести конфигурацию в .env при расширении
